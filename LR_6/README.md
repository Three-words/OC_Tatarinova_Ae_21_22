## 4. Теоретические сведения

Пузырьковая сортировка — итеративное сравнение и обмен соседних элементов массива.  
Теоретическая сложность:

| Сценарий        | Сложность |
|-----------------|-----------|
| Лучший случай   | O(N)      |
| Средний случай  | O(N²)     |
| Худший случай   | O(N²)     |
| Доп. память     | O(1)      |

## 6. Аналитическая оценка сложности

Пузырьковая сортировка имеет теоретическую сложность:

- **O(N²)** — средний и худший случай  
- **O(N)** — лучший случай (почти отсортированные массивы)

Эксперимент подтверждает квадратичный рост времени выполнения: при увеличении размера массива в 10 раз время возрастает примерно в 90–120 раз.

### Наблюдения:

1. **ArrayList и LinkedList показали почти одинаковые результаты**,  
   так как сортировка выполняется через преобразование в массив.
2. **ArrayDeque немного медленнее**, но остаётся эффективным благодаря низким накладным расходам на элементы.
3. Основная часть времени — сама сортировка пузырьком, а не преобразование структур.

## 7. Обсуждение результатов

1. Все три структуры демонстрируют квадратичную зависимость времени от размера входных данных.  
2. ArrayList и LinkedList практически совпадают по скорости.  
3. ArrayDeque показывает небольшое отставание на больших объёмах данных.  
4. Проведение 50 повторений позволило стабилизировать результаты.  
5. Пузырьковая сортировка подтверждает свою непригодность для больших массивов.
